# repo-colaborativo

Este repositorio registra las actividades que realiza un usuario, 
las guarda en una lista, y luego cuenta cuantas veces se realiza cada actividad. 
Consiste en un archivo con las funciones (funciones_habitos) y otro en donde se 
llaman dichas funciones.
